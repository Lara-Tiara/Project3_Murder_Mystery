# Project3

This project aims to implement a game called ’Arcadia Mystery’, which in-
volves role-playing and narrative quizzes, asking players to solve a fictional
murder and find out who’s the murderer among them. To handle starvation,
the game allows the players to track the reading progress of each other, which
puts additional pressure on the narrative reading period, enhancing the com-
petitive and immersive experience.
The game is text-based with a 2D user interface rather than a video game,
and will be developed with Unity using C#.
